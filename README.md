# BMS Bundle Manifest Specfication

BMS のリソース情報を主に記述するマニフェスト仕様です。  
JSON Schema で記述されています。  
BMS に紐つく本体・差分の情報等を JSON で公開するために使用できます。

詳細は実際の JSON Schema をみてください。
